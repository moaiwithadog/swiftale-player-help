## 1. プレイ画面の操作

メインプレイ画面では、毎ターン次のように行動します。

### 行動を選ぶ・自由に入力する

- **行動候補** … AIのGMが毎ターン4つの候補を出します。クリックするだけで宣言できます（選択肢式のゲームブックのような感覚）。
- **自由入力** … 画面下の入力欄に、日本語で直接行動を書いて送信できます。台詞のロールプレイやPCの心理描写を添えると、GMがそれを拾って判定や描写に活かします。

### タグchip（障害・所持品をワンタッチで入力に差し込む）

画面には現在の **障害** と **所持品** が一覧表示されています。

- 障害や所持品を**クリックすると、その名前がタグchipとして自由入力欄に挿入されます**。
- 「この障害に対して」「このアイテムを使って」といった行動を、タイプせずに素早く組み立てられます。
- 挿入したchipと自由文を組み合わせて送信できます。

### 休息

- **小休息** … HP・SP が +1 回復。連続でも取れますが、回復量は増えません。
- **大休息** … HP・SP 全回復、Tension 0。安全な場所で十分に休めるときだけ可能です。

### 状況タグ・終了宣言

ヘッダーの状況タグは、今の目的の進み具合を示します。クリックすると**セッション終了のダイアログ**を開けます（自分から区切りをつけたいとき）。

## 2. 履歴をさかのぼる（PASTモード）

画面下部の**履歴バー**で、これまでのターンを振り返れます。

- **◀ / ▶ / スライダー** … 過去のターンへ移動。ステータスや描写がそのターン時点の状態に切り替わり、「PAST T◯」バッジが表示されます。
- **LIVE ボタン** … 最新のターン（現在地）へ戻ります。

PAST モードで過去ターンを見ている間は、行動の宣言や入力はできません（閲覧専用）。やり直したい場合は、次の「世界線分岐」を使います。

---

## 3. 世界線分岐（やり直し・別の可能性）

「あの選択をやり直したい」「別の可能性を試したい」と思ったら、**世界線分岐**でその地点から新しい世界線を作れます。マルチエンディングの作品で別ルートに入る感覚に近く、**どの地点からでも分岐できる**のが特徴です。

### 操作

1. オープニング後、ライブの行動受付中、または PAST モードの任意のターン結果の後で **「世界線分岐」ボタン**を押します。
2. 新しいワールド名の編集ダイアログが出ます（初期値は `元のワールド名(分岐-日付 時刻)`）。
3. 作成後、**「新世界へ移る」**か**「今の世界を続ける」**かを選びます。
   - 「新世界へ移る」を選んだとき、分岐元のターンで使っていたAIモデルと現在の設定が違う場合は、設定確認を挟むことがあります。

分岐で作られた世界は、独立したワールドとして残ります（元の世界はそのまま）。

> **体験版・PLモードでは世界線分岐は使えません**（フル版機能）。

## 4. リプレイを書き出す

遊んだ物語は、**リプレイ**としてテキストに書き出せます。いいものができたら、どこかに公開するなど自由に使ってもらって構いません。

- **セッション単位**・**キャンペーン単位**のどちらでも書き出せます。
- Export ダイアログから**リプレイ形式**を選ぶと、プレイヤー向けの読み物として整形されたテキストが得られます。

> 開発者向けの詳細ログを含む **デバッグ書き出し** はフル版限定で、体験版・PLモードでは無効です。不具合の報告時などに使います。

## 5. 設定画面

タイトル画面の **SETTINGS**、または各画面右上の「設定」ボタンから開きます。主な設定項目は次のとおりです。

| 項目 | 内容 |
|---|---|
| **LLM プロバイダ・接続先・モデル** | Ollama / クラウド API の切り替え、接続先 URL、使用モデルの選択。詳細は [LLM セットアップガイド](./llm-setup.md)。 |
| **API キー** | クラウド API のキー登録（OS の資格情報ストアに保存。値は再表示されません）。 |
| **BGM / SFX 音量・ミュート** | BGM と効果音を、それぞれ独立したスライダーで調整。 |
| **表示設定** | 画面表示まわりの設定。 |
| **サポート導線** | 公式ヘルプサイト・サポートフォームを開くボタン。 |
| **一括初期化** | アプリ内データをすべて消して初期状態に戻す自己救済機能（次項）。 |

設定画面には「保存」「保存して戻る」「保存せず戻る」の3ボタンがあり、未保存の変更があると確認ダイアログが出ます。

> 設定はいつでも変更できます。Ollama とクラウド API の切り替えも、ここから何度でも行えます。

## 6. 一括初期化（詰んだときの最終手段）

進行不能な不具合などで**どうにも先に進めなくなったとき**の最終手段として、設定画面の末尾から**アプリ内データをすべて削除して初期状態に戻す**ことができます。

- 削除対象は、ゲームデータ（ワールド・キャンペーン・セッション）、LLM 設定・音量設定、ログ、保存済みの API キーすべてです。
- 赤枠のデンジャー表示で、誤操作を防ぐために**2段階の確認**（警告 → 確認文字列の入力）を経て実行します。
- 実行後はアプリを終了し、**手動で再起動**してください（案内が表示されます）。

> **元に戻せません。** バックアップ機能はないため、消えて困るデータがある場合は実行しないでください。
> なお体験版（PLモード）とフル版は同じ保存領域を使うため、一括初期化はフル版と共有しているデータも削除します。

## 7. 体験版・PLモードでできること／できないこと

体験版（および PL モード）は、フル版と同じデータを読み、ひととおりのゲーム動作を試せますが、**新規作成系と分岐機能はロック**されています。

| 操作 | 体験版 / PLモード |
|---|---|
| 同梱サンプルや作成済みデータで遊ぶ | ✅ できる |
| 未開始セッションの開始 / 中断セッションの再開 | ✅ できる |
| 完了セッションからの引継ぎセッション作成・プレイ | ✅ できる |
| セッション終了・キャンペーン終了・固定ルール昇格 | ✅ できる |
| リプレイ書き出し | ✅ できる |
| 設定変更・一括初期化 | ✅ できる |
| ワールド / キャンペーン / PC の新規作成・編集・削除 | ❌ フル版機能 |
| 初回（最初の）セッションの作成 | ❌ フル版機能 |
| 世界線分岐 | ❌ フル版機能 |

ロックされた操作はボタンが無効化され、ホバーで「PLモードでは利用できません」と表示されます。
