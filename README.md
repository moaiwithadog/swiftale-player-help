# SWIFTALE PLAYER

**LLM をゲームマスターにして、ひとりで遊ぶテキスト TRPG**

[![Platform: Windows](https://img.shields.io/badge/platform-Windows-blue)](https://github.com/)

> **ダウンロード**: Steam / DLSite（準備中）

---

## これは何か

SWIFTALE PLAYER は、AI（LLM）がゲームマスター（GM）役を務める、ひとりで遊ぶテキスト TRPG アプリです。

プレイヤーは **「決断と描写」だけに集中** できます。
成否の判定・ステータス計算・物語の一貫性の担保は、アプリが裏側で引き受けます。
GM はあなたの行動に応じてリアルタイムで描写を生成し、次の行動候補を提示します。

### BYOL（Bring Your Own LLM）方式

このアプリは、**プレイヤー自身が用意した AI を GM として使う** BYOL 方式を採用しています。

- 自分の PC 上で動かす **Ollama（ローカル）**
- 各社のクラウドサービスを使う **クラウド API**（Claude / OpenAI / Gemini / Grok）

どちらを選ぶかはプレイヤーが自由に決めます。
**このアプリの開発者サーバーにゲームデータや入力内容が送られることはありません。**

### ゲームの構造

物語は3つの階層で構成されます。

```
ワールド（世界観）
  └─ キャンペーン（大きな物語）
       └─ セッション（1回のプレイ）
```

キャンペーン内のセッションはひとつの時系列でつながり、前回の結果を引き継いで次の冒険へ進めます。
プレイ中に確定したことは「固定ルール」としてその世界に積み重なっていきます。

## 動作環境

| | 要件 |
|---|---|
| **OS** | Windows 10 / 11 |
| **ディスク（Ollama 使用時）** | Ollama 本体 + モデル分（例: `gemma4:e2b` で約 2GB 以上） |
| **メモリ（Ollama 使用時）** | モデルサイズに応じて異なる。`gemma4:e2b` で 4GB 以上推奨 |
| **ディスク / メモリ（クラウド API 使用時）** | 特別な要件なし |
| **ネットワーク** | クラウド API 使用時はインターネット接続が必要 |

> **Ollama を使う場合のご注意**: PC の空き容量と処理性能が必要です。「無料・準備不要」ではありません。詳しくは [LLM セットアップガイド](./llm-setup.md) を参照してください。

## はじめかた

### Step 1. アプリをインストールして起動する

ダウンロードしたインストーラを実行し、アプリを起動します。
タイトル画面が表示されたら起動成功です。

### Step 2. LLM を設定する

「START」を押すと、**初回セットアップウィザード** が開きます。

ウィザードでは、LLM の利用方式（Ollama またはクラウド API）を選んで設定します。

**迷ったときは**：
- インストール作業を避けたい → **Gemini（クラウド API）** の無料枠が最短導線
- 外部送信・API 課金を避けたい → **Ollama**（ローカル）

設定の詳細手順は **[LLM セットアップガイド](./llm-setup.md)** にまとめています。

### Step 3. サンプルセッションを遊んでみる

セットアップ完了後、「**サンプルセッションを試す**」ボタンで同梱のサンプルシナリオをすぐ遊べます。
世界観の理解やアプリの操作感をつかむための導入シナリオです。

## LLM の設定

このアプリで GM として使える LLM は2種類あります。

### Ollama（ローカル）

自分の PC 上で LLM を動かします。
ゲーム内容が外部サービスへ送信されず、API 利用料も発生しません（Ollama 自体は無料）。
ただし Ollama 本体とモデルのインストール、PC の空き容量・処理性能が必要です。

### クラウド API

各社の API を使います。
インストール作業は不要で、高品質な出力が期待できます。
ゲーム中の入力とセッション内容が選択したプロバイダのサーバーへ送信され、API 利用料が発生し得ます。

| プロバイダ | 特記事項 |
|---|---|
| Google Gemini | 無料枠あり。クラウドAPIを初めて試す方に最適 |
| Anthropic Claude | 高品質な日本語生成 |
| OpenAI | 幅広いモデル選択 |
| xAI Grok | — |

> **Gemini の無料枠について**: Google Gemini API にはリクエスト数ベースの無料枠があります。コスト不安がある方はまず Gemini から試すことをおすすめします。無料枠の上限はGoogle の方針により変わることがあるため、[Google AI Studio](https://aistudio.google.com/) で最新情報を確認してください。

各プロバイダの API キー取得手順・推奨モデル・よくある問題は  
**[→ LLM セットアップガイド](./llm-setup.md)** で詳しく説明しています。

## このゲームが使う AI について

### データの送信先

| 利用方式 | 送信先 |
|---|---|
| Ollama（初期設定） | このPC上の Ollama のみ |
| クラウド API | 選択したプロバイダのサーバー（Google / Anthropic / OpenAI / xAI） |
| 開発者サーバー | **送信しません** |

このアプリは BYOL 方式のため、**SWIFTALE PLAYER の開発者へゲームデータが送られることはありません**。
クラウド API を選んだ場合、プレイヤーの入力内容とセッション文脈は選択したプロバイダへ送信されます。
各プロバイダのプライバシーポリシーも併せて確認してください。

### ガードレール（安全ゲート）

このアプリには、不適切なコンテンツの生成を抑制するための安全ゲートが組み込まれています。

- プレイヤーの入力内容は **送信前** に検査されます
- AI の出力内容は **表示・保存前** に検査・サニタイズされます
- AI を使わないルールベースの判定で動作します

初期方針は「日本の一般的な少年向け冒険作品程度」を上限とする SFW/General です。

### AI 生成コンテンツの性質

このアプリはプレイ中にテキストをリアルタイム生成します（Live-Generated AI Content）。
生成される描写・裁定・行動候補は AI によるものであり、開発者がすべての出力を事前に確認・保証するものではありません。

## よくある質問

**Q. 完全無料で使えますか？**  
A. Ollama を使う場合、API 利用料は発生しません。ただし Ollama とモデルのインストール、PC の性能・空き容量が必要です。クラウド API では Google Gemini の無料枠を利用すると一定量まで無料で試せます。

**Q. API キーを持っていません。クラウド API は使えますか？**  
A. API キーは各社のウェブサイトで別途発行が必要です。ChatGPT Plus などの月額プランのログイン情報とは別のものです。詳しくは [LLM セットアップガイド §3](./llm-setup.md#3-クラウド-api-を使う) を参照してください。

**Q. Ollama とクラウド API は後から切り替えられますか？**  
A. はい。Settings 画面またはセットアップウィザードからいつでも変更できます。

**Q. PC のスペックが低いのですが動きますか？**  
A. クラウド API を使う場合、AI 処理はサーバー側で行われるため PC スペックに依存しません。Ollama を使う場合は、使用するモデルに応じたメモリ・ディスク容量が必要です。

**Q. ゲームデータはどこに保存されますか？**  
A. このPC上のローカルデータベースに保存されます。クラウドへの自動バックアップはありません。

**Q. 日本語以外の言語で遊べますか？**  
A. 現在は日本語を主言語として開発しています。

その他の質問は [LLM セットアップガイド §5（FAQ）](./llm-setup.md#5-よくある質問faq) も参照してください。
